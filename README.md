# cookbook-arch-remaster

Remastering Arch Linux ISO.

## Supported Platforms

Arch Linux

## Usage

### remaster::default

Include `remaster` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[remaster::default]"
  ]
}
```
